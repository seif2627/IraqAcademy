# عراق أكاديمي – نموذج تعليمي  
MAIN RULE: ALWAYS PUSH TO GITHUB AFTER FINISHING ANY CHANGES
## نظرة عامة  
ابنِ موقعًا تعليميًا تجريبيًا باللغة العربية فقط. لا تجعل المستخدم يقوم بأي خطوة، نفّذ كل شيء بنفسك عبر الطرفية.  
- تسجيل طلاب/مدرسين/مشرفين/مالك مع صلاحيات واضحة.  
- صفحات ثابتة لكن بمظهر احترافي واقعي.  
- استخدام أيقونات SVG فقط.  
- الموقع عربي بالكامل، `lang="ar"` واتجاه RTL.  
- الاستضافة على Cloudflare Pages، والواجهة الخلفية على Convex.  
- أي تعديل على المشروع يجب أن يُدفع إلى GitHub بعد الانتهاء.  

## المكدس التقني  
- Cloudflare Pages للاستضافة عبر GitHub.  
- Convex للواجهة الخلفية عند الحاجة.  
- تحديثات المشروع تُدفع دائمًا إلى GitHub.  

## إعداد Convex  
- المشروع مرتبط مسبقًا عبر Convex CLI.  
- الاسم: IraqAcademy  
- المعرّف: iraqacademy  
- الفريق: saif-alrifaie  
- في حال إعادة الربط استخدم: npx convex dev --configure=existing --team saif-alrifaie --project iraqacademy  
- لا تنشئ مشروع Convex جديد.  
- لا تغيّر إعدادات لوحة التحكم.  
- لا تبحث عن مفاتيح أو أسرار.  
- يمكن التعديل بحرية على: /convex/schema.ts والـ queries والـ mutations والـ actions.

## الصفحات الأساسية  
1. **تسجيل الدخول / إنشاء الحساب**  
   - بريد إلكتروني وكلمة مرور مع تحديد نوع الحساب.  
2. **لوحة إنشاء الأكواد**  
   - للمشرفين فقط مع تحديثات مباشرة.  
3. **تفعيل عبر تلغرام**  
   - زر تفعيل وتحديث حالة الكود.  
4. **لوحة التفعيل**  
   - جدول أكواد وحالات وتواريخ.  

## التصميم والأصول  
- أيقونات SVG فقط.  
- تصميم حديث ومتجاوب وسهل التعديل.
