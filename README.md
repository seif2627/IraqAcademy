# Iraq Academy

Iraq Academy is a modern learning platform that connects students with expert teachers and structured courses. It offers a clean, responsive experience across devices with role-based management for admins, teachers, and students.

## Highlights
- Browse courses and enroll quickly
- Manage teacher profiles and course catalogs
- Maintain student profiles and enrollments
- Responsive, professional UI
